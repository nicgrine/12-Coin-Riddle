# The Twelve Coin Riddle

A polished interactive version of the classic twelve coin logic puzzle: use three weighings to find the fake coin and determine whether it is heavier or lighter.

## How It Works

- Drag coins between the main tray, the scale, and the logic zones
- Place an equal number of coins on both pans
- Use the balance scale exactly 3 times
- Review the weighing log
- Make your final guess: which coin is fake, and is it heavier or lighter?

## Features

- Standalone single-file HTML game
- Animated SVG balance scale
- Drag-and-drop coin movement
- Persistent weighing log
- Logic zones for sorting coins
- Final reveal screen
- Responsive dark puzzle-game UI
